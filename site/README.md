# HotWell.kz — сайт

Статический сайт компании HotWell.kz (дома и домокомплекты из СИП-панелей по
Казахстану). Домен: **hotwellkz.kz**.

## Структура
```
site/
├── index.html        # Главная страница
├── css/style.css     # Дизайн-система (токены + компоненты) — ЗАФИКСИРОВАНА
├── STYLEGUIDE.md     # Правила стиля для всех будущих 40+ страниц
└── assets/           # Изображения (заменить плейсхолдеры .ph на фото)
```

## Запуск локально
Открыть `index.html` в браузере. Сборка не нужна — чистые HTML/CSS.
Для локального сервера: `python3 -m http.server` из папки `site/`.

## Как добавлять страницы
Смотри **STYLEGUIDE.md** → раздел «Шаблон новой страницы». Копировать шапку и
футер из `index.html`, собирать контент из готовых компонентов. Стиль не менять.

## Что заменить на реальные данные
- Телефон `+7 700 000 00 00` → реальные номера офисов.
- Плейсхолдеры `.ph` (зелёные блоки) → фото завода, проектов, интерьеров.
- Цены проектов и адреса офисов (Астана, Алматы).
