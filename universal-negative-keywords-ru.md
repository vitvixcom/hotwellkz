# Универсальный список минус-слов · Русскоязычный рынок (KZ/RU)

Готовый к вставке список минус-слов для сервисного/строительного бизнеса в Google Ads
на **русскоязычную аудиторию** (Казахстан, Россия и др.). Адаптация
`universal-negative-keywords.md` под русский язык + казахский мини-блок.

**Уровень: аккаунт или кампания. Тип соответствия = ШИРОКОЕ (BROAD), если не указано иное.**

> ⚠️ **Морфология русского.** Широкое минус-соответствие НЕ ловит склонения и синонимы:
> минус-слово «вакансия» не блокирует «вакансию» и «вакансий». Поэтому список включает
> основные словоформы. После запуска дочищайте по отчёту «Поисковые запросы».

> 🏗️ **Для стройки/ремонта особое внимание разделу F.** Нельзя блокировать «работы»
> (= «строительные/кровельные работы»), «проект», «под ключ», «цена/стоимость»,
> «недорого», «заказать», «рядом» — это запросы покупателей.

---

## Как пользоваться

1. **День 1:** вставьте универсальные термины (раздел A) в общий список минус-слов
   (Инструменты → Общая библиотека → Списки минус-слов).
2. **Прикрепите список к кампании** (один клик).
3. **1–2 недели** работайте с этими значениями по умолчанию.
4. **Еженедельно** открывайте отчёт «Поисковые запросы»: добавляйте новый мусор, убирайте
   слова, которые случайно блокируют покупателей.
5. Большинство аккаунтов стабилизируются на **~150–250 минус-словах** за 60 дней.

> **Качество > количество.** Одно широкое «вакансии» отсекает тысячи нерелевантных показов.

## C. Гео-минусы — заполнить под себя

Если работаете по **Астане + 50 км**, блокируйте города/страны ВНЕ зоны, которые
конкурируют за ваши ключи (житель Астаны может искать «дом в Алматы»).

> Таргетинг по присутствию (presence, 50 км) ловит большую часть, но НЕ блокирует
> жителя Астаны, который ищет «каркасные дома Алматы». Гео-минусы закрывают эту дыру.

## D. Бренды конкурентов — точечно

Обычно НЕ добавляйте бренды конкурентов в основную SKAG-кампанию (эти запросы конвертят
в 2–3 раза лучше). Добавляйте их как минусы во ВСЕ кампании, КРОМЕ отдельной «конкурентной».
Если конкурентной кампании пока нет — добавьте 5–10 локальных конкурентов широким минусом
на уровне аккаунта, чтобы не платить за бренд-мисматч.

## E. Типы соответствия — что когда

| Тип | Для чего | Пример |
|---|---|---|
| **Широкое** (по умолчанию) | Одиночный универсальный мусор | `вакансии`, `бесплатно`, `курсы` |
| **Фразовое** (`"в кавычках"`) | Словосочетания, где важен порядок | `"своими руками"`, `"как построить"` |
| **Точное** (`[в скобках]`) | Точный нерелевантный запрос без вариаций | `[купить дом в астане]` |

> ⚠️ **Русская морфология.** Широкий минус НЕ включает склонения/синонимы: минус «вакансия»
> не блокирует «вакансию»/«вакансий». Добавляйте ключевые словоформы (как в списках выше).
> Для словосочетаний берите **фразовое** — оно ловит запрос с этими словами в любом окружении.

## F. Что НЕ добавлять (частые ошибки)

Выглядят как мусор, но блокируют покупателей:

| Термин | Почему кажется минусом | Почему нельзя |
|---|---|---|
| `цена`, `стоимость`, `сколько стоит` | «просто смотрят» | поздняя воронка, высокий интент |
| `недорого`, `дёшево`, `доступный` | «ищут дешёвое» | бюджетный, но готовый купить |
| `рядом`, `рядом со мной`, `возле` | слишком общее | один из самых конверсионных модификаторов |
| `заказать`, `купить` | «изучают» | прямое намерение купить |
| `под ключ`, `проект` | общее | это ваш продукт |
| `работа` / `работы` | похоже на соискателя | в стройке = «строительные/кровельные работы» |
| `жұмыс` (каз.) | то же | то же — «құрылыс жұмыстары» |
| `отзывы` | «изучают» | сравнивающий покупатель на финальной стадии |
| `лучший` | «изучают» | сравнивающий покупатель близко к решению |

## G. Поддержка списка (еженедельно · 15 мин)

1. Google Ads → **Аналитика и отчёты → Поисковые запросы**
2. Период: последние 7 дней
3. Сортировка по расходу (по убыванию)
4. По каждому запросу:
   - **Была конверсия?** ✓ Пропустить
   - **Потрачено заметно, 0 конверсий?** ✗ Добавить как минус кампании (фразовое для словосочетаний, широкое для одиночных)
   - **Спорно (мало денег, 0 конв., но интент возможен)?** Пометить, подождать неделю
5. Добавить минусы одной операцией
6. Готово

Первые 6 недель находится 5–15 новых минусов в неделю, затем 1–3.

## H. Массовое добавление через Claude Code / API

Скрипты в репозитории:
- `integrations/google-ads/build_negative_list_ru_kz.py` — создаёт общий список
  «Универсальные минус-слова RU+KZ v1» и прикрепляет к кампании (всё BROAD, парсер
  поддерживает `"фразу"` и `[точное]`).
- `integrations/google-ads/add_campaign_negatives.py` — добавляет минус-слова на уровне
  конкретной кампании (после одобрения).
- `integrations/google-ads/find_campaign_negatives.py` — тянет поисковые запросы и
  помечает кандидатов в минусы (SOP `find-and-add-negatives.md`).

Промпт-пример:
```
Добавь русско-казахский список минус-слов как общий Shared Negative Keyword List
«Универсальные минус-слова RU+KZ v1» и прикрепи к кампании {campaign_id}.
Тип соответствия BROAD, кроме терминов в кавычках (фразовое) и скобках (точное).
Минусы: [вставить разделы A.1–A.8].
```
