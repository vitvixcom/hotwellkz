# Запуск кампании «под ключ» (SKAG) — HotWell.kz

Даёшь 3 вещи — **ключ + регион + бюджет** — скилл собирает всю кампанию на PAUSED:
кампания → гео → язык → расписание → минус-слова → 20 RSA → ассеты → аудитория +50%.

Движок: `.claude/skills/launch-campaign/launch_campaign.py` (`validate` / `build`).
Правила копирайтинга и редполитики — `anatomy-of-a-good-ad.md`,
`ad-assets-best-practices.md`, скилл `/generate-ads`.

## Рабочий процесс (что делает Claude)

### 1. Вход от пользователя
- **Ключевое слово** (напр. «каркасные дома», «сип дома», «модульные дома»)
- **Регион/город** (напр. «Астана», «Алматы»)
- **Бюджет** $/день

SKAG-ключ = `«{ключ} {город}»` (напр. «каркасные дома Астана»), фразовое соответствие.
Название кампании = `[SKAG] {Ключ} {Город}`.

### 2. Найти посадочную страницу
Слаг на сайте: `{город-транслит}-{услуга-слаг}.html` (напр. `astana-karkasnye-doma.html`,
`almaty-sip-doma.html`). Найди файл в `site/` (Glob/grep) под город+услугу:
- услуги→слаг: каркасные→`karkasnye-doma`, сип→`sip-doma`, модульные→`modulnye-doma`,
  панельные→`panelnye-doma`, дом из сэндвич→`dom-iz-sendvich-paneley`, дача→`dachnye-doma`.
- Если нашёл — `final_url = https://hotwellkz.kz/{slug}.html`. Сверь H1 (должен совпадать с темой).
- Если нет — спроси пользователя URL или предложи каталог `proekty.html`.

### 3. Гео (координаты города для проксимити 50 км)
Подставь координаты центра города (микроградусы вычисляет движок):
Астана 51.1605,71.4704 · Алматы 43.2389,76.8897 · Шымкент 42.3417,69.5901 ·
Караганда 49.8047,73.1094 · Актобе 50.2839,57.1670 · Тараз 42.9000,71.3667 ·
Павлодар 52.2873,76.9674 · Усть-Каменогорск 49.9489,82.6279 · Семей 50.4111,80.2275 ·
Атырау 47.0945,51.9238 · Костанай 53.2198,63.6354 · Кызылорда 44.8488,65.4823 ·
Уральск 51.2333,51.3667 · Петропавловск 54.8753,69.1628 · Актау 43.6410,51.1980.
Если города нет в списке — оставь `geo` без lat/lon, только `"city"`: движок найдёт
гео по названию через geo_target_constant.

### 4. Сгенерировать копию (ты — копирайтер, по правилам /generate-ads)
Заполни `rsa_pools` грамотным русским (склонения!):
- **pinned** (3–5): варианты ключа с гео — «{Ключ} {Город}», «{Ключ ед.ч.} в {Город предл.}»
  (напр. «Каркасные дома Астана», «Каркасный дом в Астане», «Каркасные дома в Астане»).
- **free** (≥28, лучше ~28): сервис-нейтральные + по 6 паттернам (оффер, доверие,
  срочность, гарантия, призыв, выгода): «Дом под ключ за 30 дней», «Фундамент входит в
  цену», «Гарантия на дом 1 год», «Расчёт стоимости онлайн», «Без скрытых доплат»,
  «Своё производство», «Опыт с 2018 года», «Тёплый дом для зим РК» и т.п. Длина ≤30.
- **descriptions** (≥20): углы услуга+скорость / доверие+цена / отличие / призыв, ≤90.
Уникальность 20 объявлений обеспечивает ротация пулов (движок). `n_ads`: 20.
Ассеты: 8 sitelinks (на реальные якоря посадочной: `#calc`,`#includes`,`#works`,`#tech`,
`#reviews`,`#contacts`,`#about` + `proekty.html`), 12 callouts, 2 snippets (Услуги/Типы).
path1=город, path2=услуга (≤15).

### 5. Записать spec.json, проверить, показать, собрать
1. Запиши spec во временную папку (формат — в шапке `launch_campaign.py`).
2. `python .claude/skills/launch-campaign/launch_campaign.py validate spec.json` → исправь ✗.
3. **Покажи пользователю** итог (кампания/ключ/бюджет/гео/посадочная/примеры объявлений)
   и дождись «ок».
4. Загрузи креды (GOOGLE_ADS_* env) и
   `python .claude/skills/launch-campaign/launch_campaign.py build spec.json`.
5. Проверь результат через API и отчитайся (Campaign ID + ссылка). Напомни, что всё PAUSED.

## Что движок делает сам (дефолты, можно переопределить в spec)
- Аккаунт `8802382037`, язык русский, расписание 08:00–22:00.
- Исключение всех стран, кроме Казахстана (анти-VPN/боты).
- Минус-слова кампании: 15 стандартных RU (+ модульные/контейнер).
- Прикрепляет общий список «Универсальные минус-слова RU+KZ v1».
- Прикрепляет warm-pixel аудиторию «Warm pixel · all visitors · hotwellkz.kz · 540d» с +50%
  (Observation). Аудитория и конверсии уже работают на всём сайте.
- Bidding: Максимум конверсий; Search-only; всё **PAUSED**.

## Заметки
- Ничего не включаем — запуск (PAUSED→ENABLED) делает пользователь.
- Бизнес-имя/логотип не добавляются автоматически (нужна проверка рекламодателя) —
  по запросу `add_business_identity.py`.
- Если бюджет < $20/день — предупреди, что Smart Bidding учится медленно.
