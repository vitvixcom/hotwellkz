# Генерация SEO-статьи для блога HotWell.kz

## Что делает
Скрипт `generate.py` через OpenAI:
1. Придумывает (или принимает) актуальную SEO-тему о СИП-панелях/строительстве в Казахстане.
2. Пишет статью **≥1500 слов** на русском: вступление, разделы `<h2>/<h3>`, списки,
   блок «Часто задаваемые вопросы», вывод с призывом к действию, внутренние ссылки.
3. Генерирует **3 изображения** (gpt-image-1): 1 главное (1536×1024) + 2 в теле (1024×1024),
   фотореалистичные дома из СИП-панелей с пейзажем под город (Алматы — горы, Астана — степь и т.д.),
   у каждой картинки **свой alt** с ключевым словом. Формат webp (быстрая загрузка = плюс к SEO).
4. Прописывает SEO-разметку: `<title>`, meta description/keywords, canonical, Open Graph,
   Twitter Card, JSON-LD `Article`.
5. Создаёт `site/blog/<slug>.html` и добавляет карточку в секцию «Блог» на `site/index.html`
   (при первой статье убирает пустое состояние).

## Требования
- Переменная окружения **`OPENAI_API_KEY`** (или файл `.openai_key` рядом со скриптом — он в .gitignore).
- Доступ в интернет (через прокси — уже настроен).
- Необязательные env: `OPENAI_TEXT_MODEL` (по умолч. `gpt-4o`), `OPENAI_IMAGE_MODEL`
  (`gpt-image-1`), `OPENAI_IMAGE_QUALITY` (`high`), `SITE_BASE_URL` (`https://hotwellkz.kz`).

## Как запускать (шаги для агента)
1. Убедиться, что задан `OPENAI_API_KEY` (`[ -n "$OPENAI_API_KEY" ] && echo ok`).
   Если нет — сообщить пользователю, как добавить (env окружения или файл `.openai_key`).
2. Запустить из корня репозитория:
   ```bash
   python3 .claude/skills/generate-article/generate.py            # тему придумает сам
   python3 .claude/skills/generate-article/generate.py --topic "Сколько стоит дом из СИП-панелей в Алматы в 2025"
   ```
   Флаг `--no-images` — сгенерировать только текст (для отладки, без расхода на картинки).
3. Пересобрать локальное превью (встроить css и картинки работ в один html), показать пользователю.
4. Проверить валидность HTML, при желании пользователя — закоммитить и запушить
   (`site/blog/<slug>.html`, новые картинки в `site/assets/blog/`, изменённый `site/index.html`,
   и `published.json` скилла) в рабочую ветку.

## Заметки
- `published.json` хранит список уже опубликованных тем — скрипт передаёт их модели,
  чтобы не повторяться.
- Можно запускать регулярно (например, по расписанию) — каждый запуск = новая статья.
- Стоимость одного запуска ≈ несколько центов за текст + ~$0.4–0.6 за 3 картинки в высоком качестве
  (зависит от тарифов OpenAI). Снизить: `OPENAI_IMAGE_QUALITY=medium`.
